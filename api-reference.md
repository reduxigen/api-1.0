# Actions

When you create an `action`, Reduxigen manages the reducer and action-creator for that `action`. Reduxigen ships with five different `actions`:

1. [update](/api-reference/update.md)
2. [asyncUpdate](/api-reference/asyncupdate.md)
3. [action](/api-reference/action.md)
4. [genericAction](/api-reference/genericaction.md)
5. [asyncAction](/api-reference/asyncaction.md)
